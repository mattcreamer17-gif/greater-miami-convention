# GMCVB Landing Page

A bespoke one-page memo built for the Greater Miami Convention & Visitors Bureau. The aesthetic echoes the Find Your Miami campaign system: sunset orange primary, ocean teal, magenta accent, warm off-white ground, Poppins display paired with Open Sans body. The page is designed to feel like the Bureau's own marketing team produced it, not a vendor one-pager.

Structure:

1. Sticky header with a Miami sun mark and the "Miami and Miami Beach" wordmark
2. Hero with the section 7 headline, eyebrow, lede, and the five Find Your Miami pillars
3. Opening memo from section 7
4. The Eduba frame (60 / 30 / 10) with a callout card on where the leverage sits
5. Neighborhood tile grid reinforcing the pillars-as-context-architecture argument
6. Simple Lyfe Stays case, with scope language only (no pricing)
7. ICM paper pointer with a live GitHub link
8. Partner posture including the mandatory NLP Logix credibility phrasing
9. Closing CTA for a 30 minute call with Matt Creamer, Calendly linked
10. Footer with "Prepared by Eduba. eduba.io"

Intended URL path: /for/greater-miami-convention-and-visitors-bureau

Assets note: the brand pack assets folder was empty at build time, so the header uses a CSS/SVG sun mark rendered inline. When the official logo file is available, drop it into this directory and replace the `.brand-mark` SVG and `.brand-text` block in index.html with an `<img>` tag pointing at it.
